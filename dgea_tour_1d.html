Description:

FastQC is one of the preferred tools for quality control. This tool analyses FASTQC data, and produces graphs to allow fast inspection of sequencing data sets.

Trim Galore! is a wrapper around the Cutadapt data preprocessing tool, which offers an easy to use interface, as well as options for removing biased methylation positions for reduced representation bisulfite sequencing (RRBS) data sets.
