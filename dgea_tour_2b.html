Description:

HISAT2 (Hierarchical Indexing for Spliced Alignment of Transcripts) is a successor of the HISAT alignmer, and is optimized for memory usage and speed. This tool is therefore recommended for large data sets.
HISAT2 is able to process single- and paired-end reads in FASTQ-format, and works with whole-genome, exome, and transcriptome data.
